Description

This website is divided in 4 parts: Presentation, Portfolio, Contact and Footer. Each part required specific technologies. This website is hosted with a node server on a Raspberry Pi.

Technologies used

  • HTML
  • CSS
  • JavaScript
  • JQuery
  • Three.js
  • Node.js

My Role

In this project, I assumed the roles of full-stack developer and designer. I also manage the installation of a Raspberry Pi as a server.

Difficulties

  • Setting a Rapsberry Pi as a server.
  • Making the website responsive.
  • Implementing all interactions and animations.

My Solutions

This website is hosted on a Raspberry Pi using Apache as a reserve proxy server. This allows me to use Node.js for the server side. To make the website responsive, I learned more about Bootstrap and media queries. Finally for the animations, I learned Three.js and new CSS tricks.

Notable Features

  • 3D background
  • Parallax effects
  • Server side with Node.js
  • Hosted on a rapsberry pi
